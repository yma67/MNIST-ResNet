# MNIST-ResNet
Classify a modified version of MNIST using ResNet

Please Read 'writeup report.pdf'

## Instructions
1. Download dataset
```
    kaggle competitions download -c comp-551-w2019-project-3-modified-mnist
```
to current directory, such that
```
---- Parent Directory
   |
   |---- project3.py
   |
   |---- test_images.pkl
   |
   |---- train_images.pkl
```
2. Run project3.py
